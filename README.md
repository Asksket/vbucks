# Fortnite V-Bucks Automation via Apex Token

This Python script automates the process of:
1. Navigating to the Apex Legends 1000 Coins page on Xbox.
2. Adding the token to cart and purchasing it.
3. Launching Fortnite via Xbox Cloud Gaming.
4. Redeeming the gifted Battle Pass token for 950 V-Bucks via Epic API.
5. Submitting a refund request for the Apex token purchase.

⚠️ Requirements:
- A configured Chrome or Edge browser profile with Tampermonkey running.
- VPN should be toggled off before Epic Games login.
- Epic authorization code must be manually pasted when prompted.
